# EMS Documentation Project

## Project Status
🚧 This EMS documentation project is currently being written and will be posted soon.

The upcoming manual will provide a detailed step-by-step guide on how to properly complete a BLS transport chart within the ambulance setting using the Zoll EMSCharts system.

## What the Manual Will Cover

The guide will include detailed instruction and walkthroughs for:

- Page 1 Documentation
- Patient Page
- Pages 2, 3, 4, 5, 8, and 9
- Billing procedures
- Uploading required documents
- Understanding and using dropdown menus
- Best practices for accurate chart completion

## Platform

This manual is being developed specifically for:

- Zoll EMSCharts  
- https://zoll.emscharts.com

## Purpose

The goal of this project is to create a clear, structured, and easy-to-follow reference manual for EMS professionals completing BLS transport documentation. The manual is intended to improve chart accuracy, workflow efficiency, and understanding of the EMSCharts platform.

More updates and content will be added soon.
